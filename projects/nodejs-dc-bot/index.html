Node.js discord bot

Overview

In this projects I created a Discord bot with the help of node.js. With this bot you are able to write !joke to get a dad joke fetched from an API. The main feature is that you are able to use the command !toggle-dad-joke to toggle a mode for the current channel to send dad-jokes everyday at midnight.
